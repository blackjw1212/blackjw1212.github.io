FORSCAN REFERENCE

Focus Mk3.5 FORScan 觀察台

整理 Ford Focus Mk3.5（2015-2018 小改款）以 FORScan 可由使用者調整的項目，涵蓋便利、儀表、影音與保養服務功能。每張卡片以標籤標出四個維度：模組、修改方式、驗證度與風險。本頁為靜態觀察參考，資料來自社群交叉比對，並非保證值；實際數值逐車不同，操作風險自負。

5主題分類
20+可調項目
Mk3.5 / 2015-2018適用車型

操作前務必先看

  • 動手前對每個模組（BCM / IPC / APIM）先「儲存 As-Built 備份」成檔，出問題即寫回。
  • As-Built 十六進位數值逐車不同，不可照抄網路數值，否則可能亮故障燈或致功能異常。
  • 能用「Central Configuration 選單」做的，就不要改 As-Built；多數設定與服務功能需 FORScan Extended License（免費）。

Global Open / Close

鎖車雙閃喇叭關閉

Double Honk

Foglamp w/o restriction

行駛中自動上鎖

Drive-away lock

起步達一定車速後自動落鎖。Mk3.5 上 BCM 選單未必有此項，社群多改用 FoCCCus 或視選單而定。

保養 · 服務功能

Service Functions · 6

機油壽命重置

Oil Life Reset

正常車輛用「鑰匙 ON + 同時踩油門與煞車約 20-30 秒」即可歸零；若改過 IPC 設定導致選項消失，FORScan 的作用是把 IPC「Oil Life Reset」功能重新啟用回來。

IPC服務 / 選單多源確認風險低

換電瓶 BMS 電池監控重置

Battery Monitor Reset

換新電瓶後必做，讓充電系統重新學習新電瓶；Mk3.5 確認支援。

PCM / BCM服務多源確認 (Mk3.5)風險低

DPF 強制再生 + 歸零

DPF Static Regen

柴油碳微粒濾清器手動 / 靜態再生，並重置學習值與壓差值。僅柴油車（如 1.5 / 1.6 TDCi）。

PCM服務僅柴油風險中

TPMS 胎壓感測器重學

TPMS Relearn

換胎或更換感測器後重學 / 寫入感測器 ID，熄滅胎壓警示。

BCM服務多源確認風險低

節氣門 / KAM / 感測器適應重置

Reset Adaptations / KAM

清除長期學習值（節氣門、MAF、O2、EGR、燃油等），換零件或怠速異常後使用。

PCM服務多源確認風險中

PATS 防盜鑰匙編程

PATS Programming

新增 / 清除鑰匙，需 Extended License。

PATS服務多源確認風險中
